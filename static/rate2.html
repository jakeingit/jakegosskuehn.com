This is a pretty basic game that plays text-to-speech voice bits when a player taps on bubbles that appear.

I make the point that you can input what text-to-speech gets played. I'v done something similar, and kids seem to LOVE it.

To play a game where it shouts out affirmation to the child makes the kid stupidly happy. Its cute.
You can can input sounds to play when a bubble gets popped. These can be set in future releases to be played completely 'uniformly' randomly or the top ones will be played more often than the bottom ones.

You will also be able to select from several text-to-speech voices. There are a few out there that we can use.
So the game design is just bubbles of differnt sizes and colors that pop up. The game will be structured behind a grid. It will randomly select a coordinate and create a bubble there.

Then the player would tap on each bubble, and it would play the text-to-speech associated with it.

If you've ever had fun with text-to-speech making silly sounds. This gameafies it to make children oddly happy. They love random variables, they just don't know what its called yet.
