I was a college kid. I did the whole lifestyle and learned a lot. I was the person at parties always introducing new drinking games to my drunken comrades. This app is a collection of what I think are the funnest drinking/party games out there.

Its more of a small informational page of drinking games. I will probally post a web version of this too.

Here we see a scrolling list of all the games in the database. If you see a game you want in here... I'll be sure to add it if you tell me. This list isnt all inclusive. I have much more.
Tapping brings you to the page. You'll see the rules and everything. The take away besides how to play it is that it was written by someone who knows the game inside out. This write up was hastily writen, but I would maybe make it easy for a drunk student to read and learn.

At the top is a rate page. You can rate the game as a fun game to play to encourage others to play it as well.
The rules continued... I had fun writing this up.
End of the rules. There is a cultural variant section here. Some games go by different names or have different house rules.
